## Usage

```
haru init <name>
```


Stores the current time under name.


```
haru days <name> 
```


Shows the time elapsed since the saved time for name.

> [!TIP]
> If the name is omitted, it will default to `default`.
>


```
haru all
```

Displays all saved time entries.

```
haru clean
```

Removes all saved time entries.
